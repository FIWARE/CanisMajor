Run

To run CanisMajor via docker, use: docker run quay.io/fiware/canismajor A full setup, including the context broker, a development-mock eth-node, the used contract and Vault as a wallet can be setup via: docker-compose -f it/docker-compose/docker-compose-env.yaml -f it/docker-compose/docker-compose-java.yaml up
